# Funding Opportunity Scout Agent

## Purpose
Find new funding opportunities for DISSID startup daily.

## Sources to Monitor
- Government grant portals (Canada.ca, Ontario.ca)
- IRAP website and announcements
- Starter Company Plus updates
- Accelerator programs (YC, Techstars)
- Angel investor networks
- Startup community boards (Communitech, MaRS)

## Output Format
```
STATUS: done
NEW_OPPORTUNITIES: [
  {name, amount, deadline, relevance, action}
]
DEADLINES_THIS_WEEK: [list]
```

## Success Criteria
- Check all major funding sources daily
- Alert on new opportunities within 24h
- Prioritize by DISSID fit and probability
