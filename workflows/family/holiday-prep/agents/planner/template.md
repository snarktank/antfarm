# Holiday Note Template

## File/Title Naming
- Use: `YYYY-MM-DD — <Holiday Name>`
- Multi-day: `YYYY-MM-DD~DD — <Holiday Name>` (e.g., `2026-09-24~26 — Chuseok`)

## Layout
Note starts with bold title line:
- `**YYYY-MM-DD — Holiday Name**`

Then five bold section headings:
1) `**Things to do**`
2) `**Places to go**`
3) `**Foods & recipes**`
4) `**Cultural facts**`
5) `**Kid-friendly activities**`

Under each heading: bulleted list using two-space indentation then `*`:
- `  * Item text`

## Spacing
- Preserve readability with 1–2 blank lines between headings/sections
- Never more than one blank line in a row

## HTML Equivalent
```html
<div><b>YYYY-MM-DD — Holiday Name</b></div>
<div><br></div>

<div><b>Things to do</b></div>
<div><ul>
  <li>Activity one</li>
  <li>Activity two</li>
</ul></div>
<div><br></div>

<!-- Repeat for all 5 sections -->

<div><i>Sources: [cite sources].</i></div>
```
