How It Works

From workflow definition to coordinated execution

1

Define Your Workflow

Create a workflow.yml that specifies your agents, their roles, and how they hand off work. Each agent gets its own workspace with custom instructions.

id: feature-dev
name: Feature Development
agents:
  - id: lead
    role: Plans and coordinates the feature
  - id: developer  
    role: Implements the feature
  - id: reviewer
    role: Reviews the code
steps:
  - agent: lead
    prompt: "Plan the implementation"
  - agent: developer
    prompt: "Implement the plan"
  - agent: reviewer
    prompt: "Review the changes"
2

Install the Workflow

Antfarm provisions agent workspaces, wires them into OpenClaw, and sets up the cron orchestrator. One command does it all.

antfarm workflow install ./workflows/feature-dev

# Or install from GitHub
antfarm workflow install https://github.com/team/workflows/tree/main/feature-dev
3

Run the Workflow

Start a workflow run with a task title. The orchestrator kicks off the first agent and tracks progress through each step.

antfarm workflow run feature-dev "Add user authentication"

# Check status anytime
antfarm workflow status "Add user authentication"
4

Agents Collaborate

Each agent does its part and signals completion. The orchestrator monitors via cron, spawning the next agent when the previous step finishes.

Quickstart

Get up and running in minutes

Prerequisites

You'll need OpenClaw installed and configured.

Install Antfarm

npm install -g antfarm

Install a Workflow

# Use the built-in feature-dev workflow
antfarm workflow install antfarm/workflows/feature-dev

# Or create your own workflow.yml and install it
antfarm workflow install ./my-workflows/custom-flow

Run Your First Task

antfarm workflow run feature-dev "Build the settings page"

# The orchestrator will spawn agents and coordinate the work
# Check status anytime:
antfarm workflow status "Build the settings page"

Commands Reference

Everything Antfarm can do

antfarm workflow install <source>

Install a workflow from a local path, GitHub repo, or URL

antfarm workflow run <id> <task>

Start a workflow run with the given task title

antfarm workflow status <task>

Check the status of a workflow run

antfarm workflow update <id>

Update a workflow to the latest version

antfarm workflow uninstall <id>

Remove a workflow and clean up its agents

antfarm workflow uninstall --all

Remove all Antfarm-installed workflows
